# soemdsp-sandbox presets

Patch preset JSON files for the browser modular environment.

- `default.json`: current default startup patch captured as editable JSON.
- `useruisettings.json`: bundled default UI/workspace settings for a clean first load.
- `useruisettings.js`: generated browser fallback wrapping the same payload as `useruisettings.json`; keep the payloads identical.
